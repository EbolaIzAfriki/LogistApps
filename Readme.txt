Для разворачивания системы в локальной сети необходимо:
  1. на машине-сервере создать базу данных MS SQK из скрипта расположенного в папке ServerDiplom;
  2. ввести имя сервера в datacontext в строке connectionstring располеженной в конфигурации приложения;
  3. в инициализаторе главной формы необходимо указать IPv4 адрес машины-хоста;
  4. в клиентском приложении указать адрес машины-хоста в методе "Connect" расположенном в DiplomAdminEditonBeta/DiplomAdminEditonBeta/TCPModels/ClientTCP.cs;

После выполения вышеописанных действий система будет готова к работе.
